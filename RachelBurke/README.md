# Http Server with simple Persistence

## Description
GET and POST requests on /simple

### How to Use:
Enter node index.js in command line. Visit http://localhost:3000/simple or or curl http://localhost:3000/simple npm i --save-dev superagent-cli superagent localhost:3000/notes post '{"noteBody":"this is some simple body content"}'

## devDependencies
* chai
* chai http
* gulp
* gulp-eslint
* gulp-mocha
* mocha

## devs
* fs
* http

## How to test
Run gulp in the Command Line. Make sure no open instances of the server are going.
